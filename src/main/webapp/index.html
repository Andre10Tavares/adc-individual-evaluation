Welcome to ADC Individual Evaluation mini help page!

Here are the operations available:

Create Account - Create new account

Login - Login on one account

Show Users - Show all existing users

Delete Account - Deletes one account

Modify Account Attributes - Modify attributes of one account

Show Authenticated Sessions - Show authenticated sessions.

Show User Role - Show role of one user

Change User Role - Change role of one user

Change User Password - Change password

Logout - Logout on one account

Clean Tokens - Cleans expired tokens

This is to help with the inputs on, for exemple, Postman. All operations are POST.
