Operation 6 - Show Authenticated Sessions

Show all authenticated sessions that are not expired.

Endpoint: /showauthsessions

Complete: https://individual-evaluation-491717.oa.r.appspot.com/rest/showauthsessions

JSON:

{
  "input": {},
  "token": { ... }
}

Token must be complete and valid.

All tokens are valid for 15 minutes.

Exemple JSON:

{
  input": {},
  "token": {
    "tokenId": "abc",
    "username": "banana@ups.com",
    "role": "ADMIN",
    "issuedAt": 0,
    "expiresAt": 900
  }
}

Authorized roles:

ADMIN

Possible Errors:

Invalid token - Error 9903

Token expired - Error 9904

Unauthorized - Error 9905

Forbidden - Error 9907
