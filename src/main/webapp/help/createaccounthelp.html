Operation 1 - Create Account

Creates a new account.

Endpoint: /createaccount

Complete: https://individual-evaluation-491717.oa.r.appspot.com/rest/createaccount

JSON:

{
  "input": {
    "username": "string@gmail.com",
    "password": "string",
    "confirmation": "string",
    "phone": "string",
    "address": "string",
    "role": "USER | BOFFICER | ADMIN"
  }
}

Gmail, just because it happened, it can be anything as long as it has an @ symbol, as if it were an email address.

Exemple JSON:

{
  "input": {
    "username": "banana@ups.com",
    "password": "pass123",
    "confirmation": "pass123",
    "phone": "111",
    "address": "lisboa",
    "role": "ADMIN"
  }
}

Authorized roles:

USER

BOFFICER

ADMIN

Possible Errors:

User already exists - Error 9901

Invalid input - Error 9906
