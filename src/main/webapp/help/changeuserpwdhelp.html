Operation 9 - Change User Password

User can change password

Endpoint: /changeuserpwd

Complete: https://individual-evaluation-491717.oa.r.appspot.com/rest/changeuserpwd

JSON:

{
  "input": {
    "username": "string@gmail.com",
    "oldPassword": "string",
    "newPassword": "string"
  },
  "token": { ... }
}

Gmail, just because it happened, it can be anything as long as it has an @ symbol, as if it were an email address.

Token must be complete and valid.

All tokens are valid for 15 minutes.

Exemple JSON:

{
  input": {
    "username": "banana@ups.com",
    "oldPassword": "pass123",
    "newPassword": "123pass"}
  },
  "token": {
    "tokenId": "abc",
    "username": "banana@ups.com",
    "role": "ADMIN",
    "issuedAt": 0,
    "expiresAt": 900
  }
}

Authorized roles:

USER

BOFFICER

ADMIN

Possible Errors:

Invalid credentials - Error 9900

Invalid token - Error 9903

Token expired - Error 9904

Unauthorized - Error 9905

Forbidden - Error 9907
