Operation 5 - Modify Account Attributes

Modify phone or address of one account

Endpoint: /modaccount

Complete: https://individual-evaluation-491717.oa.r.appspot.com/rest/modaccount

JSON:

{
  "input": {
    "username": "string@gmail.com",
    "attributes": {
      "phone": "string",
      "address": "string"
    }
  },
  "token": { ... }
}

Gmail, just because it happened, it can be anything as long as it has an @ symbol, as if it were an email address.

Token must be complete and valid.

All tokens are valid for 15 minutes.

Exemple JSON:

{
  input": {
    "username": "banana@ups.com",
    "attributes": {
      "phone": "123",
      "address": "madeira"
    }
  },
  "token": {
    "tokenId": "abc",
    "username": "banana@ups.com",
    "role": "ADMIN",
    "issuedAt": 0,
    "expiresAt": 900
  }
}

Authorized roles:

USER

BOFFICER (also can modify attributes of USER)

ADMIN (can modify attributes of any role)

Possible Errors:

User not found - Error 9902

Invalid token - Error 9903

Token expired - Error 9904

Unauthorized - Error 9905

Invalid input - Error 9906

Forbidden - Error 9907
