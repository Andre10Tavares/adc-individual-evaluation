Operation 2 - Login

Login on one account.

Endpoint: /login

Complete: https://individual-evaluation-491717.oa.r.appspot.com/rest/login

JSON:

{
  "input": {
    "username": "string@gmail.com",
    "password": "string"
  }
}

Gmail, just because it happened, it can be anything as long as it has an @ symbol, as if it were an email address.

All tokens are valid for 15 minutes.

Exemple JSON:

{
  "input": {
    "username": "banana@ups.com",
    "password": "pass123"
  }
}

Authorized roles:

USER

BOFFICER

ADMIN

Possible Errors:

Invalid credentials - Error 9900

User not found - Error 9902
