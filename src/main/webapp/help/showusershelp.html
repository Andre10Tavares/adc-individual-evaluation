Operation 3 - Show Users

Show all existing users

Endpoint: /showusers

Complete: https://individual-evaluation-491717.oa.r.appspot.com/rest/showusers

JSON:

{
  "input": {},
  "token": { ... }
}

Token must be complete and valid.

All tokens are valid for 15 minutes.

Exemple JSON:

{
  "input": {},
  "token": {
    "tokenId": "abc",
    "username": "banana@ups.com",
    "role": "ADMIN",
    "issuedAt": 0,
    "expiresAt": 900
  }
}

Authorized roles:

BOFFICER

ADMIN

Possible Errors:

Invalid token - Error 9903

Token expired - Error 9904

Unauthorized - Error 9905
