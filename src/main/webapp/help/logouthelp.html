Operation 10 - Logout

Logout on one account

Endpoint: /logout

Complete: https://individual-evaluation-491717.oa.r.appspot.com/rest/logout

JSON:

{
  "input": {
    "username":"string@gmail.com"
  },
  "token": { ... }
}

Gmail, just because it happened, it can be anything as long as it has an @ symbol, as if it were an email address.

Token must be complete and valid.

All tokens are valid for 15 minutes.

Exemple JSON:

{
  input": {
    "username": "banana@ups.com"
  },
  "token": {
    "tokenId": "abc",
    "username": "banana@ups.com",
    "role": "ADMIN",
    "issuedAt": 0,
    "expiresAt": 900
  }
}

Authorized roles:

USER

BOFFICER

ADMIN (can logout anyone but will logout all sessions of that user)

Possible Errors:

Invalid token - Error 9903

Token expired - Error 9904

Unauthorized - Error 9905

Forbidden - Error 9907
